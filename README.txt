# Test1BNN
Prueba 1 de examen tecnico BNN
Para que se pueda ejecutar correctamente
Generar un archivo con nombre test.txt en C:// con las palabras a validar si son palíndromos
En caso de no subir el archivo la aplicación lo solicitara y se tendra que ejecutar nuevamente.
La aplicacion muestra las palabras del archivo y si son palíndromos o no.
La aplicacion ignora si encuentra una palabra solo con espacio.
